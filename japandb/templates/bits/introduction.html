Most Japanese websites use the same core database for their information from kanji readings to example sentences.
The point of this website is to build a new database with more detailed and (constantly) updated information.
This information should help users read NHK Web Easy articles.
The website content is generated by crawling those articles, so please read their articles and support them!
This website does not come with translations. This means you should use plugins like Rikai-chan or Rikai-kun for translating words.

This website has no ads and is provided entirely for free.

Firefox users: Remember to download HTML Ruby so that the furigana will render properly

We hope to be your learning resource of choice.
